SlowView Keyboard shortcuts:
============================

Function        Hotkey:                 Menuitem:
========        =======                 =========

Actual Size     NUMBLOCK *              View->Actual Size
Zoom In         NUMBLOCK +              View->Zoom In
Zoom Out        NUMBLOCK -              View->Zoom Out
Next Image      PageDn/Space            View->Image->Next
Previous Image  PageUp/Backspace        View->Image->Previous
First Image     Home                    View->Image->First
Last Image      End                     View->Image->Last
Browse          Return/Enter,           File->Browse
Exit            Escape                  File->Exit
Help          	F1                      Help->Contents
Refresh		F5			Edit->Refresh
Slideshow       Ctrl+Shift+S		Configuration->SlideShow->Start/Stop
					
Open            Ctrl+O,                 File->Open...
Save as         Ctrl+S                  File->Save as
Reload          Ctrl+R                  File->Reload
Print           Ctrl+P                  File->Print
Undo            Ctrl+Z                  Edit->Undo
Copy            Ctrl+C                  Edit->Copy
Cut             Ctrl+X                  Edit->Cut
Paste           Ctrl+V                  Edit->Paste
Crop            Ctrl+D                  Edit->Crop
Select All      Ctrl+A                  Edit->Select All
Fullscreenmode  Ctrl+F (Alt+Return)	View->Fullscreen
Always Maximize Ctrl+M                  View->Always Maximize Window
TileImage       Ctrl+T			View->Image as Tilemap

Save            Shift+S                 File->Save
Rename          F2 (Shift+R)           	Edit->Rename...
Copy to         Shift+C                 Edit->Copy to...
Move to         Shift+M                 Edit->Move to...
Flip            Shift+F                 Image->Flip
Mirror          Shift+G                 Image->Mirror
Rotate left     Shift+H                 Image->Rotate 90°left
Rotate right    Shift+J                 Image->Rotate 90°right
Delete File     Delete                  Edit->Delete File
		+ SHIFT	-> Delete to trash always

Scroll Image	Cursor Keys
		+ SHIFT -> Scroll slow
		+ CONTROL -> Scroll fast

Next/Prev Media Mousewheel		
		+ CONTROL -> ZoomIn/ZoomOut
