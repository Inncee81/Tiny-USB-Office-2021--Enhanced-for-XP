Capturing Image 

Capturing image can be done in two ways: 

  1. By using the popup menu and clicking on "Take single image". There will be a delay as set in the Video options. Quickly move the cursor to the designated location and wait for the signal that the image has been captured; 
  2. By using the shortcut key. Image will be captured immediately. 

When an image is taken the AviScreen icon will turn into a camera animation with flash. 

When the "Follow the cursor" mode is not activated and AviScreen is not in full screen mode, a red dotted rectangle will also be shown indicating the area that has been captured.
