Adding other Root Files

Choose Arrange editing mode.

To add the new root file at the end of the project, click on “Add video”. (To add at the start or insert into the project see Adding a new video track.

The following options are available:

Fast or Slow Motion Video

Adds a video clip that has been set to run at either fast or slow motion.

1/. Select “Fast/Slow-motion video”
2/. Click “OK”
3/. Click on “Set” to locate, select and open video file.
4/. Click on “Fit” and then “OK” to determine duration of selected video file.
5/. Adjust duration to make the video run faster or slower. (shorter = fast motion, longer = slow motion)
6/. Click on “OK” to add fast or slow motion video. 

Please Note: Fast or Slow Motion videos are added without a soundtrack.

Music

Adds a music clip (.WAV file) as the base of a project. This is ideal for building a film up around a song or a piece of music or for making your own video clip.

1/. Select “Music-clip base”.
2/. Locate, choose and open music file.
3/. Click OK to add.

Please note: to add video to a music-clip base, use “Add effect”/ “Overlays”/ “Movie overlay”

Black or Colored Frames

Inserts black or colored frames (useful as a background for overlays and effects)

1/. Select “Black frames”. Click “OK”.
2/. Set duration and color.
3/. Click “OK” to add.

Running Still Images as a Slideshow

Allows you to run a series of still images as a slideshow.

1/. Select “Bitmap Clips”. Then click “OK”.
2/. Click on “Set”, then locate, select and open the file of Bitmap images you wish to use as a Bitmap Clip. (Please note: you should resize the images first to fit the movie you are making.)
3/. Set the number of images to appear per second. 
4/. Click “OK” to add.

Adding a Group

Allows you to add a group in order to simplify a project.

1/. Select “Group”. Click “OK” 2x.

Once you have added a group, you can simplify your project by moving video files into the queue in the group. You can then add effects at the level of the group. For instance, when you want them to take effect across more than one video file.

Adding a Sub-project

Allows you to add an AviTricks project file as a sub-project (sealed) and include it in your own project.

1/. Select “Sub-project”. Click “OK”.
2/. Use “Set” to locate, select and open AviTricks project file. Click “OK”.

See: Working with Projects and Sub-projects.
