TERMS

.AVI – Audio Video Interleaved – files are video files that conform to Microsoft Resource Interchange File Format – RIFF – specifications. AVI file names end with the extension “.avi”.

AviSplit is a Bobyte program for splitting, rejoining and clipping .AVI files. It is available in both a basic freeware and an upgraded shareware version. Read more about AviSplit or download it.

Bitmap A bitmap is a graphic image made up of pixels (picture elements). Each pixel or "bit" contains color information for the image. Bitmap formats include BMP. JPG and GIF.

Bitmap Clip A bitmap clip is a folder of bitmap files played at a set rate. Something like a slide show.

.WAV The standard sound format for Windows. AviTricks can export sound from video as a .WAV file. There are many programs available for converting other sound formats into .WAV.
