AviTricks Basics: Layering Effects

When you use two or more effects simultaneously, the order of application of the effects can change the appearance of the end product. For instance, if you add mirror effect and TV-shop in that order, you will see multiple images of a mirror effect image. If you change the order to first TV-shop and then mirror image, the multiple images themselves will be reflected and the image on the individual “TV screens” will be standard.

You can change the order of the images using the Tree Structure.
