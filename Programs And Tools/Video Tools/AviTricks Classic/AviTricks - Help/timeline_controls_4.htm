Timeline controls IV

Setting and changing the starting and ending points of a time-track (mark-in and mark-out) dominates the whole process of video editing. AviTricks is optimized for this task. 

Changing or setting mark-in and mark-out  is only available under Arrange mode. 

First Method: Using pegs.

Pegs are two points that designate the range over which an effect will be added.  

Double-click on the bar to add a peg. Since only two pegs can appear on one bar at a time, a prompt may appear asking the user to first confirm the removal of existing pegs. 

Pegs may be removed from a bar through popup menu "Peg\Clear pegs". Another method to remove a peg is by double-clicking on it.

When a bar has a pair of pegs, an effect added to this bar will begin and end (mark-in and mark-out) according to the position of the pegs.

Second Method: Using left mouse in combination with [Ctrl] key.

To shift the starting point of an effect, point the mouse at the starting point of the effect while pressing the [Ctrl] key.

To shift the end point of an effect, point the mouse at the end point of the effect while pressing the [Ctrl] key.

To shift the effect without changing its duration, point the mouse to the middle of the bar while pressing the [Ctrl] key.

Third method: Using the popup menu.

Right-click with the mouse on the point (inside the bar) where the mark-in should take place. Move the mouse slowly to menu "Mark-in and Mark-out", and then on to "Set as Mark-in point". Click there. Repeat this process to set the mark-out point.

You can release a mark-in point using "Release Mark-in point". This menu will be disabled when no Mark-in point is set for this effect. When the Mark-in has been released, the effect will start at the starting point of its parent. Using "Release the mark-in" is faster than dragging it using the Second method mentioned above.

Fourth Method: Using viewer for marking.

This feature is accessible through the popup-menu, deep under  "Mark-in and Mark-out" and only relevant to a video. 

Using the viewer you can set the mark-in and mark-out visually. To set the Mark-in, drag the tracker to the position that you want, then click the button with the red exclamation mark inside the "Custom start box".

To review the settings, press "locate" button at any time.
