Filters, Controls, Special Effects 
Brightness Adds an adjustable brightness setting.
Light Temperature Adjustment Allows you to make the picture “warmer” or “colder” by adding an adjustable intensity setting for individual colors. Set color first and then level. To apply this effect to more than one color simultaneously, add it once for each color setting.
Filters: Sharpen, Soften, Emboss, Blur, Edge, Laplace, Enchange, Sketch Various special effects filters. See preview screen. Some have a fixed intensity level, others are adjustable.
Mirror Puts a mirror in the middle of the screen, allows you to choose left or right reflection.
Negative Shows the image in negative.
Stepback Reverses the action to create a stepback effect. (Sound track unaffected.)
Freeze Freezes the action for the duration of the effect. (Sound track unaffected.)
