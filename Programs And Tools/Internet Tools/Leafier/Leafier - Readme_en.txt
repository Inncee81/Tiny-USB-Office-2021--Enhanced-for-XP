Overview
--------
Leafier is a simple HTML editor with tag insertion feature.
Features are as follows:
- Supported character encodings are UTF-8, UTF-16 (Little-Endian,
  Big-Endian), ISO-8859-1 (Latin-1), Shift_JIS, EUC-JP, ISO-2022-JP.
- Tabbed multi document interface, syntax highlighting
- Tag, attribute, property, etc are selectable by Ctrl+Click.
- All registration tags can be modified.
- Other features are font settings, color settings, key customization,
  bookmark, launch external tool, etc.
