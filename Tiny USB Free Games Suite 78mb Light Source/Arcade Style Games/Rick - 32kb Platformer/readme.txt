Technical details
The Rick32Kb needs DirectX8.0. All the game use real textures and polygons. (well, strange for a 2d game huh ?). That version looks like the ATARI-ST version, and has additionnal features:

A demo mode !
High resolution filtered texture ( Toggle with "F" key )
An ATARI-ST soundchip emulator ( ST-Sound engine )
12 minutes of real ATARI music

Commands
Cursor key to move Rick ( up, down, left & right )
CONTROL key + Up to fire a gun bullet
CONTROL key + Down to set a bomb.
"F" to toggle filtered graphics ( enable by default )
"W" to toggle wireframe mode ( just to see you buy a powerfull 3d card to play poor 2d game :-))
