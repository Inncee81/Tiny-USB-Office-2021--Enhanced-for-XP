The Sensor Scan

The sensor scan is an extremely useful item which provides information on other objects in the galaxy. It has four settings: off, min, med and max. You can control the setting by using the arrows in the Command Console, or using the - and + keys on the keyboard.

At all settings above off, the sensor scan will provide you with information on most objects that are close enough to appear in the Main Viewer. For instance, beside every starship will be displayed its energy and torpedoes remaining.

Objects that you see on the Long Range Scan (LRS) depend on the Sensor Scan setting:

  • MIN: displays all planets, starbases and uncloaked vessels..

  • MED: displays all planets, starbases, uncloaked vessels and asteroids. Asteroids show up simply as dots.

  • MAX: displays all planets, starbases, uncloaked vessels, asteroids and most cloaked ships. (though the cloaked ships tend to flicker)

See "The Long Range Scan" in this help guide to see how the different objects are represented on the LRS.
